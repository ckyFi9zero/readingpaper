# readingpaper
This repository records the papers that I read carefully.
The goal is to track my reading progress, organize important research papers, and build a structured knowledge base.

For each paper, I record:

- Reading date
- Publication year
- Venue / journal
- Ranking or level (e.g., CCF / JCR)
- DOI
- PDF file stored in this repository

The PDF versions of papers are stored in the `pdf/` folder.

------

# Repository Structure

```
readingpaper/
│
├─ pdf/            # stored PDF papers
├─ notes/          # optional detailed notes for each paper
└─ README.md       # reading log
```

# Notes

Detailed notes for some papers may be written in the `notes/` folder.

Example:

```
notes/
├─ paper_name1.md
├─ paper_name2.md
```

These notes may include:

- Problem definition
- Key ideas
- Method overview
- Experiments
- Personal understanding
